=======================================
PyLadies Handbook
=======================================

本家PyLadiesに自分の地域でPyLadiesを立ち上げるためのマニュアル ` PyLadies Handbook <PyLadies Kit — PyLadies Organizer Handbook>`_ があります。

.. image:: _static/images/pyladies_handbook_1.png

書いてあること
=======================================

- ローカルPyLadiesの作り方
- PyLadiesの運営方法
- 誰が参加できるかなど
- PyLadiesの行動規範
- ロゴなどの権利

pip
---------

pipでもインストールできます。

.. code-block:: bash
   :linenos:

   $ pip install pyladies
   $ pyladies handbook
